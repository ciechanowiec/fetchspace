How to Use

  1. Start the Bot

    Open Fetch Space Bot on Telegram and start the bot.

  2. Upload Your File

    In the chat input area, select File as the input type and send it to the bot. Besides File as the input type, the bot also accepts videos from the gallery, music, and voice messages.

  3. Get Your Link

    The bot will respond with a public link to the file you uploaded, such as https://fetchspace.org/api/assets.download.41d66272-51c0-4511-9b07-30e69e0c62ff.zip. Anyone who has the link can anonymously, without login or registration, download the file.

  4. Manage Your Files

    Want to deactivate your links? Issue a /clear command to delete all files you've uploaded to the bot. Once deleted, the links to deleted files will no longer work.
